# Simple Docker Stats

A simple way to get basic stats about your containers' performance. Backed by `docker stats` and `awk`

## Usage

```
Usage: dstats.sh [command] (options)

Commands:
  run:         run the daemon
  show:        return the stats

Options:
  -p|--prefix: log directory prefix

```
